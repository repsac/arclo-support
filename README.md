# Arclo Support

**Arclo** is a one-tap swing arcade game for iPhone and iPad. You control a pair of orbs joined by a tether. Tap to swap which orb is the anchor, fling the pair forward, and thread a run of glowing gates while a rising void chases you down. It ships with two distinct physics modes, ten difficulty tiers, an original synth soundtrack, and Game Center leaderboards.

## About

Arclo is an independent game by Ed Caspersen. Gameplay runs on SpriteKit with a fixed-timestep physics loop; the shell, menus, and soundtrack player are SwiftUI. The soundtrack is original instrumental synthwave, and all sound effects are short procedural tones (replaceable with bundled audio).
